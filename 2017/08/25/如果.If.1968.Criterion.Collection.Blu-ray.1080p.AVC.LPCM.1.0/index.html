◎简　　介

寄宿学校的新学年开始了，高年级学生麦克整天心神不宁。麦克从杂志上撕下画片贴在墙上，和好友约翰尼与华莱士一起饮酒聊天。一天下午，同学们正在观看橄榄球赛，麦克和约翰尼偷偷溜出学校，骑着偷来的摩托车，去咖啡馆和女招待调情嬉笑。他们回到学校，受到校方的鞭打。麦克等人怀恨在心，决心报复学校的粗暴与野蛮。军训中，麦克用枪和刺刀威胁教士，校方罚他去清理大厅下面的废品。无意中，麦克他们发现了一些遗弃的枪支弹药。讲演日那天，校方请来丹森将军给学生和来宾讲演。突然，大厅的下面冒出浓烟，学生和来宾混乱起来，跑到院子里，遭到麦克和—伙同学的射击。。

◎幕后制作

本片根据戴维·舍文和约翰·霍雷特的电影剧本《反叛者》(Crusaders)改编。本片导演林赛·安德森1967年拍摄了引起争议的《白色公共汽车》(The white bus)后再次开火。他在本片中对英国寄宿生制度进行了猛烈抨击，寄宿学校里到处充满着虚伪和偏见，在受人尊重的正直的伪装下，存在着最严重的独断专横和非人的压迫。导演为了能充分说明寄宿制度的虚伪而又避免引起不必要的政治争论，在影片中巧妙地使用幻想与虚构的情节，使影片成了“不真不假”，“亦真亦假”的故事。影片片名只是“写在墙上的一些东西”，信与不信，就看你自己。影片结构复杂，讽刺尖锐，导演技巧高超，安排巧妙。本片在1969年第22届戛纳国际电影节上获得金棕榈大奖。
